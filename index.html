PRECISION DISRUPTER UPDATER


HOW TO UPDATE


  1. Make sure you're using Chromium or the latest Chrome Desktop
  2. Unplug the Precision Disrupter from your Eurorack power supply
  3. Connect the Precision Disrupter to your computer via a USB-C cable while holding the DFU button located next to the USB-C socket
  4. Click UPDATE and select the STM device from the pop-up menu
  5. Wait until the process is completed and enjoy your new firwmare :)


OS Release Date: 20 OCTOBER 2022


New on this OS update:


  • ADC completely DMA-based
  • UI smoothness improved
  • Overall timing is tighter
  • CV input trigger-type signals are now detected down to 5% dutycycle at 120 BPM and 10% at 320 BPM. (synced disruptions are now working)
  • Preset buttons don't light up red anymore if there are unsaved changes - The the recall button still lights up if an unsaved change is present on the current preset.
